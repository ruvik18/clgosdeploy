# Institution Logos Folder

Place your engineering college/institute logo images in this directory to display them in the app.

## File Naming Convention

Name the logo image file to match the college's `slug` (in lowercase, hyphenated) with a `.png` extension:

- IIT Bombay → `iit-bombay.png`
- IIT Delhi → `iit-delhi.png`
- IIT Madras → `iit-madras.png`
- IIT Guwahati → `iit-guwahati.png`
- IIT BHU Varanasi → `iit-bhu-varanasi.png`
- NIT Trichy → `nit-trichy.png`
- NIT Surathkal → `nit-surathkal.png`
- NIT Warangal → `nit-warangal.png`
- IIIT Hyderabad → `iiit-hyderabad.png`
- BITS Pilani → `bits-pilani.png`
- DTU Delhi → `dtu-delhi.png`

## Specifications

- **Format**: `.png`
- **Dimensions**: Square aspect ratio (e.g. `128x128px` or `256x256px`) works best.
- **Background**: Transparent background is highly recommended.
